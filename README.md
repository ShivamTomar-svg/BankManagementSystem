# 💻 Bank Management System

## 📌 Overview

This project is a **Java-based Bank Management System** designed to simulate core banking operations such as account creation, transactions, and balance management.
It demonstrates the use of **Object-Oriented Programming, database connectivity, and GUI development**.

---

## 🚀 Features

* 🔐 User Registration & Login System
* 🏦 Account Creation and Management
* 💰 Deposit and Withdrawal Operations
* 📊 Balance Inquiry
* 🗄️ Secure Data Storage using MySQL

---

## 🛠️ Tech Stack

* **Java (Core + OOP Concepts)**
* **JDBC (Database Connectivity)**
* **MySQL (Database Management System)**
* **Swing (GUI Development)**

* Create a MySQL database
* Update database credentials in code

### 4. Run Application

* Execute main class (`main_Class.java`)
